Image Denoising & Portfolio Optimization

基于 Lecture 5 & Lecture 6/9 (Lin, JNU) · 预计阅读 20 分钟

应用一 / Application I: Image Denoising / 图像去噪

Digital image basics / 数字图像基础

Type / 类型Per pixelTypical use
Binary / 二值1 bit (black/white)Text scans, fingerprints / 文本扫描、指纹
Grayscale / 灰度8 bits (0–255)X-rays, document scans / X光、文档扫描
RGB / 真彩色24 bits (3×8)Color photographs / 彩色照片

Vectorization / 向量化

Grayscale image is an m×n matrix V ∈ ℝ^{m×n}.
Flatten into a vector / 拉直为向量：vec(V) ∈ ℝ^{mn}
This converts image processing into a vector-space optimization problem.

Denoising optimization model / 去噪优化模型

Let f = observed noisy image, u = unknown clean image.
minᵤ   ½‖u − f‖₂² + τ·φ(u)

• ½‖u−f‖₂²（data fidelity term / 数据保真项）：don't stray too far from observation
• φ(u)（regularization term / 正则化项）：encodes prior knowledge (e.g., natural images are smooth)
• τ（regularization parameter / 正则化参数）：τ large → more smoothing but may blur details; τ small → faithful but noisy

→ Solved by FPPA since φ is often non-smooth (e.g., ℓ₁ norm for piecewise smoothness).

应用二 / Application II: Portfolio Optimization / 投资组合优化

Markowitz Mean-Variance Model (OMV) / Markowitz 均值-方差模型

N assets, w = portfolio weights (what fraction to invest in each).
min_w   wᵀΣw   (minimize portfolio variance = risk / 最小化组合方差（风险）)
s.t.   wᵀμ = ρ    (target expected return / 目标收益率)
     wᵀ1ₙ = 1   (fully invested / 全额投资，权重和为1)
     w ≥ 0      (no short selling / 不可卖空)

This is a quadratic program / 二次规划：quadratic objective + linear constraints.
Connects to our Lagrange multipliers (equality constraints) and inequality handling.

Sparse & Stable Markowitz Portfolio (SSMP)

min_w   1/T·‖Rw − ρ1_T‖₂² + τ‖w‖₁
s.t.   wᵀμ̂ = ρ, wᵀ1ₙ = 1

• ℓ₂ term: sample-based risk (avoids estimating Σ)
• ℓ₁ term ‖w‖₁: encourages sparsity / 鼓励稀疏性 — most wᵢ = 0, invest in only a few assets
→ More practical: real portfolios don't hold 500 stocks!
→ Solved by FPPA (ℓ₁ is non-smooth, needs proximity operator).
ℓ₁ vs ℓ₂ regularization / ℓ₁ 与 ℓ₂ 正则化对比：
• ℓ₂ (ridge): shrinks coefficients but none become exactly zero — all assets held
• ℓ₁ (LASSO): non-differentiability at zero induces sparsity — most weights become exactly zero
→ Both image denoising and portfolio optimization exploit this property of ℓ₁.
→ FPPA is the algorithmic workhorse for solving these composite (smooth + ℓ₁) problems.
