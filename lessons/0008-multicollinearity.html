0008 · 多重共线性
Multicollinearity

基于 Lecture 8/10 (Lin, JNU) · 预计阅读 20 分钟

1. What is multicollinearity? / 什么是多重共线性？

Perfect multicollinearity / 完美共线性：
∃ a ≠ 0 such that Xa = 0 → rank(X) < K+1 → |XᵀX| = 0 → (XᵀX)⁻¹ does not exist.

Approximate multicollinearity / 近似共线性（现实中常见）：
Xa ≈ 0 → |XᵀX| ≈ 0 → (XᵀX)⁻¹ exists but extremely unstable.
Intuition / 直观理解：
If you use both "height in cm" and "height in inches" to predict weight,
the two variables are almost perfectly correlated → the model can't tell which one is doing the work.
用「身高(cm)」和「身高(吋)」同时预测体重，两个变量几乎完全线性相关，模型无法区分各自贡献。

2. Why is it harmful? / 为什么有害？

Causal chain / 因果链：
X columns nearly linearly dependent / X 的列近似线性相关
→ |XᵀX| ≈ 0 (determinant near zero / 行列式接近零)
→ (XᵀX)⁻¹ has huge diagonal entries / 逆矩阵对角元爆炸
→ D(b̂) = σ²(XᵀX)⁻¹ → var(β̂ⱼ) extremely large / 方差极大
→ Estimates unstable, t-statistics insignificant, signs may flip! / 系数不稳定，t 检验不显著，符号甚至可能反转

3. Three Equivalent Characterizations / 三种等价表征 ⭐ 考试常考

Theorem / 定理：以下三条等价 (equivalent)：
(i) Columns of X exhibit (approximate) multicollinearity / X 列近似线性相关
(ii) XᵀX has an eigenvalue ≈ 0 / XᵀX 存在接近 0 的特征值
(iii) |XᵀX| ≈ 0 / 行列式接近零

Proof sketch / 证明思路：
XᵀX is symmetric positive semi-definite → orthogonally diagonalizable: XᵀX = UΛUᵀ
Λ = diag(λ₁,…,λₙ), λ₁ ≥ … ≥ λₙ ≥ 0.
(i)→(ii): Xa≈0 → aᵀXᵀXa≈0 → zᵀΛz≈0 (z=Uᵀa≠0) → ∃λⱼ≈0.
(ii)→(iii): |XᵀX| = Πλᵢ, λⱼ≈0 → product ≈0.
(iii)→(ii): Πλᵢ≈0 → at least one λⱼ≈0 → corresponding eigenvector u gives Xu≈0.

4. VIF / 方差膨胀因子 (Variance Inflation Factor)

For bivariate regression (two independent variables):
var(β̂₁) = σ² / [Σ(x₁ᵢ−x̄₁)² · (1 − r₁₂²)]
where r₁₂ = sample correlation between x₁ and x₂.

VIF₁ = 1/(1 − r₁₂²)
As r₁₂² → 1 (high correlation) → VIF → ∞ → var(β̂₁) → ∞

Rule of thumb / 经验法则：VIF > 10 (i.e., r² > 0.9) → serious multicollinearity / 严重共线性。

5. Remedies / 补救方法

  • Drop variables / 删除变量：remove redundant predictors
  • Ridge regression / 岭回归：β̂ = (XᵀX + λI)⁻¹Xᵀy — stabilize by adding λ to diagonal (biased but lower variance)
  • Principal Component Regression / 主成分回归：project correlated X onto uncorrelated principal components
  • Increase sample size / 增大样本量：more data reduces (but doesn't eliminate) the problem
