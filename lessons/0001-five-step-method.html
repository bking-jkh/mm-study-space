0001 · 五步建模法与单变量优化
Five-Step Method & One-Variable Optimization

基于 Lecture 1 (Lin, JNU) & Meerschaert §1.1–1.2 · Glossary / 术语表 · 预计阅读 30 分钟

1. 什么是数学建模？
What is Mathematical Modeling?

数学建模就是把一个现实问题翻译成数学语言（translate a real-world problem into mathematical language）， 用数学工具求解（solve it with mathematical tools），再把结果翻译回现实答案（translate the result back into plain language）。

Key Understanding / 关键理解：数学建模不是「套公式」。每一道现实问题都是独一无二的， 你首先要决定用什么数学结构来描述它（这是最难的一步），然后再求解。

Mathematical modeling is the art of translating problems from an application area into tractable mathematical formulations. — Meerschaert, 4th Edition

2. 卖猪问题 / The Pig Problem

Problem statement / 问题原文：
A pig weighing 200 pounds gains 5 pounds per day and costs 45 cents a day to keep. The market price for pigs is 65 cents per pound, but is falling 1 cent per day.
When should the pig be sold to maximize profit?

一头猪现在重 200 磅，每天增重 5 磅，每天饲养成本 45 美分。 当前市场价是 65 美分/磅，但每天跌 1 美分。什么时候卖利润最大？

热身：硬币问题 / Warm-up: The Coin Problem

Problem / 问题：There are 11 coins, nickels (5 cents) and dimes (10 cents), totaling 70 cents. How many of each?
有 11 枚硬币，全是 5 分（nickel）和 10 分（dime），总价值 70 分。各有几枚？
  1. 你意识到需要两个变量：nickels 的数量（叫它 x）和 dimes 的数量（叫它 y）→ determine the variables / 确定变量
  2. 两个条件：x + y = 11（total count）和 5x + 10y = 70（total value）→ write equations (assumptions) / 写出方程（假设）
  3. 目标是「求出 x 和 y」→ state the objective / 明确目标
  4. 解方程组：x = 8, y = 3 → solve the model / 求解
  5. 回答：8 nickels and 3 dimes → answer the question in plain language / 回答问题

这 5 个步骤，就是五步建模法（Five-Step Method）的雏形。

3. 五步法详解：卖猪问题
The Five-Step Method Applied to the Pig Problem

The five steps are / 五步依次为：

  1. Ask the question / 提出问题
  2. Select the modeling approach / 选择建模方法
  3. Formulate the model / 列出模型
  4. Solve the model / 求解模型
  5. Answer the question / 回答问题

Step 1: Ask the Question / 提出问题

Step 1 有三个子任务（three sub-tasks）：列出变量（variables）、写出假设（assumptions）、明确目标（objective）。

① Variables / 变量

Symbol / 符号 Meaning / 含义 Units / 单位 Why? / 为什么需要它？
t time / 时间 days 核心决策变量（when to sell）
w weight of pig / 猪的体重 lbs 体重随时间变化，影响收入
p price per pound / 单价 $/lb 价格随时间变化
C cost of keeping pig / 饲养总成本 $ 成本是利润的减项
R revenue from sale / 卖出收入 $ Revenue = price × weight
P profit / 利润 $ Profit = Revenue − Cost（我们要最大化的量）
常见错误 / Common Pitfall：混淆变量（variables，会变化的量）和常量/参数（constants/parameters，在当前设定中不变的值，如初始体重 200 lbs，增重速率 5 lbs/day）。

技巧：给每个变量标注单位（label units）。如果后续计算中单位对不上，假设就有误。

② Assumptions / 假设

Assumption 1 / 假设 1：猪的体重线性增长（linear weight gain）
w = 200 + 5t
Initial weight is 200 lbs, gains 5 lbs per day.

Assumption 2 / 假设 2：猪价线性下降（linear price decline）
p = 0.65 − 0.01t
Current price is $0.65/lb, falls by $0.01 per day.
用「−」号表示下降（negative sign = decline）。

Assumption 3 / 假设 3：饲养成本每天固定（constant daily cost）
C = 0.45t
Cost is 45¢ = $0.45 per day × t days.

Assumption 4 / 假设 4：收入 = 单价 × 重量
R = p × w   (Revenue = price × weight)

Assumption 5 / 假设 5：利润 = 收入 − 成本
P = R − C   (Profit = Revenue − Cost)

Assumption 6 / 假设 6：时间不能为负（physical constraint）
t ≥ 0   (You cannot sell the pig in the past)
Why write out all assumptions? / 为什么要把假设写清楚？
因为假设很可能不准确（assumptions may be inaccurate）。猪真的每天增重 5 磅吗？价格真的每天跌 1 美分吗？ 如果不明确写出假设，后面就没法做灵敏度分析（sensitivity analysis）——判断你的结论有多可靠。

③ Objective / 目标

Objective: Maximize P (profit) / 最大化利润 P
用数学符号：max P(t), subject to t ≥ 0

目标必须写清楚是「最大化（maximize）」还是「最小化（minimize）」什么量。

Step 2: Select the Modeling Approach / 选择建模方法

观察 Step 1 的成果：只有一个独立变量 t，目标函数 P(t) 只依赖 t，约束只有 t ≥ 0。

这符合单变量无约束优化（one-variable unconstrained optimization）的标准形式。

What is one-variable optimization? / 什么是单变量优化？
给你一个函数 y = f(x)，定义在集合 S 上（如 x ≥ 0），找 x 取什么值时 f(x) 最大（或最小）。

核心定理 / Key Theorem：
If f attains its maximum or minimum at an interior point x ∈ S, and f is differentiable at x, then f′(x) = 0.
如果 f 在 S 的内部某点 x 处可微且取得最大值/最小值，那么在那一点，导数为零。

直觉理解 / Intuition：想象一座光滑的山。山顶（maximum）和山谷底（minimum）都是平的，斜率（slope）= 0。导数 f′(x) 就是斜率。

⚠️ 注意：f′(x) = 0 只是必要条件（necessary condition），不是充分条件（not sufficient）。f′(x)=0 的点可能是：
  • 极大值点 / local maximum（山顶）
  • 极小值点 / local minimum（山谷底）
  • 拐点 / inflection point（先上后继续上，短暂停顿）
另外，极值也可能出现在定义域的边界（boundary）处（如 t=0），边界点不需要 f′(x)=0。

Step 3: Formulate the Model / 列出模型

把 Step 1 的具体问题代入（substitute）Step 2 的标准形式，消去中间变量。

Step 1: Write P in terms of intermediate variables
P = R − C

Step 2: Substitute R = p × w and C = 0.45t
P = p × w − 0.45t

Step 3: Substitute p = 0.65 − 0.01t and w = 200 + 5t
P = (0.65 − 0.01t) × (200 + 5t) − 0.45t

Step 4: Expand (分配律展开)
(0.65 − 0.01t)(200 + 5t)
= 0.65×200 + 0.65×5t − 0.01t×200 − 0.01t×5t
= 130 + 3.25t − 2t − 0.05t²
= 130 + 1.25t − 0.05t²

Subtract 0.45t / 减去 0.45t：
P = 130 + 1.25t − 0.05t² − 0.45t
= 130 + 0.8t − 0.05t²

Standard form / 标准形式（降幂排列 / descending powers）：
P(t) = −0.05t² + 0.8t + 130

Final model / 最终模型：
Maximize f(x) = −0.05x² + 0.8x + 130, subject to x ≥ 0
（这里 t → x，P → f，是数学惯例 / notation convention）
Sanity check / 验证你的模型是否合理：
• t = 0（sell immediately / 立即卖）：P = 130 → 利润 $130
• t = 8（我们怀疑是最优解）：P = −0.05×64 + 0.8×8 + 130 = 133.2
• t = 20（wait too long / 等太久）：P = −0.05×400 + 0.8×20 + 130 = 126
利润先增后减（profit goes up then down）→ 存在一个最优时间点 ✓

Step 4: Solve the Model / 求解模型

① Take the derivative and set it to zero / 求导并令其为零

f(x) = −0.05x² + 0.8x + 130

f′(x) = −0.1x + 0.8   （x² → 2x, 所以 −0.05x² → −0.1x；0.8x → 0.8；130 → 0）

Set f′(x) = 0 / 令导数为零：
−0.1x + 0.8 = 0 → 0.1x = 0.8 → x = 8

② Verify it's a maximum, not a minimum / 验证这是极大值

Method 1 / 方法一：Second derivative test / 二阶导数检验
f″(x) = −0.1 < 0 for all x
Second derivative negative → f is concave down（开口向下的抛物线）→ x=8 is the global maximum / 全局最大值。

Method 2 / 方法二：First derivative sign analysis / 一阶导数符号分析
When x < 8: f′(x) = −0.1x + 0.8 > 0 → f is increasing / 递增
When x > 8: f′(x) = −0.1x + 0.8 < 0 → f is decreasing / 递减
So x = 8 is the peak / 所以 x=8 是峰值。

③ Compute the maximum profit / 计算最大利润

f(8) = −0.05 × 64 + 0.8 × 8 + 130
    = −3.2 + 6.4 + 130
    = 133.2

Maximum profit = $133.20, achieved at day 8.

Step 5: Answer the Question / 回答问题

The golden rule of Step 5 / Step 5 的黄金法则：
Rephrase the results in nontechnical terms. Avoid mathematical symbols and jargon.
用非技术语言重新表述结果，避免数学符号和术语。

Anyone who can understand the original question should be able to understand your answer.
任何能理解原始问题的人都应该能理解你的答案。
The pig should be sold after 8 days, yielding a net profit of approximately $133.20.
第 8 天卖出这头猪能获得最大利润，约为 $133.20。

This conclusion is valid only if our assumptions remain valid: the pig continues to gain 5 lbs/day, the price continues to fall by 1¢/day, and the daily cost remains 45¢/day.
前提是假设成立：猪继续每天增重 5 磅，价格继续每天跌 1 美分，饲养成本保持每天 45 美分。

4. 灵敏度分析 / Sensitivity Analysis

4.1 为什么要做灵敏度分析？/ Why Sensitivity Analysis?

建模始于假设（assumptions）。回头看看哪些假设是确定（certain）的，哪些是不确定（uncertain）的：

  • 猪的当前体重 / current weight（200 lbs）→ 很确定 / very certain
  • 当前猪价 / current price（65¢/lb）→ 很确定 / very certain
  • 饲养成本 / cost per day（45¢/day）→ 很确定 / very certain
  • 增重速率 / growth rate g（5 lbs/day）→ 有点不确定 / somewhat uncertain
  • 猪价跌速 / price falling rate r（1¢/day）→ 非常不确定 / very uncertain

灵敏度分析回答：If a parameter deviates from our assumption, how much does our conclusion change?

4.2 方法一：逐点试探 / Brute-force Approach

对猪价跌速 r（price falling rate，原假设 r = 0.01 $/day）：

rChange / 变化Optimal x / 最优 xChange in x / x 的变化
0.008−20%15.0 天+87.5%
0.009−10%11.1 天+38.8%
0.010baseline / 基准8.0 天baseline / 基准
0.011+10%5.5 天−31.3%
0.012+20%3.3 天−58.8%

r 只变化了 ±20%，但最优卖出时间从 3.3 天变到 15 天。模型对 r 非常敏感 / the model is very sensitive to r。

4.3 方法二：公式推导 / Systematic Approach ⭐ 考试重点

核心思路：把参数当作未知字母，推导出最优解关于该参数的公式。

对猪价跌速 r 做灵敏度 / Sensitivity to price falling rate r

Modify assumption / 修改假设：p = 0.65 − rt（把 0.01 换成 letter r）

Re-derive profit / 重新推导利润函数：
f(t) = (0.65 − rt)(200 + 5t) − 0.45t
    = 130 + 3.25t − 200rt − 5rt² − 0.45t
    = −5rt² + (2.8 − 200r)t + 130

Take derivative / 求导：
f′(t) = −10rt + (2.8 − 200r) = 0
→ x(r) = (2.8 − 200r) / (10r) = 0.28/r − 20

Verify / 验证：r = 0.01 → x = 0.28/0.01 − 20 = 28 − 20 = 8 ✓

⚠️ Domain check / 定义域检查：
x(r) ≥ 0 ⟺ r ≤ 0.014
If r > 0.014, the optimal strategy is sell immediately (x=0) / 立即卖出，
because f′(t) < 0 for all t ≥ 0（每一天利润都在降）。

计算灵敏度系数 / Computing the Sensitivity Coefficient

Definition / 定义：Sensitivity S(x, p) = (Δx/x) / (Δp/p)
x 的相对变化（relative change）÷ p 的相对变化（relative change）

With calculus / 用微积分表示（let Δp → 0）：
S(x, p) = (dx/dp) × (p/x)

Apply to our case / 应用于猪问题：
x(r) = 7/(25r) − 20
dx/dr = −7/(25r²)
At r = 0.01: dx/dr = −7/(25 × 0.0001) = −2800

S(x, r) = (−2800) × (0.01/8) = (−2800) × 0.00125 = −3.5
How to interpret S(x, r) = −3.5? / 怎么解读？
• Sign / 符号：negative → r increases ⟹ x decreases（猪价跌越快，越早卖）
• Magnitude / 大小：3.5 → a 1% increase in r causes ≈ 3.5% decrease in x
• |S| = 3.5 > 2 → highly sensitive / 高度敏感，r must be estimated accurately

Rule of thumb / 经验法则：
• |S| < 0.5: insensitive / 不敏感（rough estimate OK）
• 0.5 ≤ |S| ≤ 2: moderately sensitive / 中等敏感
• |S| > 2: highly sensitive / 高度敏感（need precise estimate）

对增重速率 g 做灵敏度 / Sensitivity to growth rate g

Modify: w = 200 + gt（原来 g = 5）

f(t) = (0.65 − 0.01t)(200 + gt) − 0.45t = −0.01gt² + (0.65g − 2.45)t + 130
f′(t) = −0.02gt + (0.65g − 2.45) = 0
→ x(g) = (0.65g − 2.45) / (0.02g) = 32.5 − 122.5/g

dx/dg = 122.5/g², at g = 5: dx/dg = 122.5/25 = 4.9
S(x, g) = 4.9 × (5/8) = 3.0625

Also sensitive / 也是高度敏感的：g 增 1%，最优时间增约 3%。

目标函数灵敏度 / Sensitivity of Optimal Profit S(y, g)

Substitute x(g) back into profit function / 把 x(g) 代回利润函数：
y(g) = 150.0625/g + 10.5625g + 50.375
dy/dg = −150.0625/g² + 10.5625
At g = 5: dy/dg = 4.56
S(y, g) = 4.56 × (5/133.2) = 0.171

Key contrast / 关键对比：
S(x, g) ≈ 3.06 → sensitive for when to sell（对决策时间敏感）
S(y, g) ≈ 0.17 → insensitive for profit（对最终利润不敏感）

This is good news: even if you misjudge the growth rate and sell at a suboptimal time, your profit won't be much lower! / 即使你判断错了增重速率、决策时间不完美，实际利润差不了多少！

5. Summary / 总结

Five-Step Method / 五步法清单

Step 1 Ask the question List variables (with units), state assumptions (equations/inequalities), state objective (maximize/minimize what)
Step 2 Select the approach Identify problem type (one-variable optimization? multivariable? linear regression?)
Step 3 Formulate the model Substitute Step 1 into Step 2's standard form → pure math problem
Step 4 Solve the model Use calculus / Newton's method / graphical methods
Step 5 Answer the question Plain language, no math jargon, state the assumptions

Sensitivity Analysis / 灵敏度分析要点

  • S(x, p) = (dx/dp) × (p/x) — dimensionless ratio of relative changes
  • Sign: positive = same direction, negative = opposite direction
  • Magnitude: |S| > 2 → sensitive; |S| < 0.5 → robust
  • Prioritize uncertain parameters（优先分析不确定度高的参数）
  • Decision sensitivity S(x,p) and objective sensitivity S(y,p) can differ greatly
