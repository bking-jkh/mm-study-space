0002 · 灵敏度分析专题练习
Sensitivity Analysis — Worked Examples & Practice

基于 Lecture 1 §2 (Lin, JNU) & Meerschaert §1.2 · 预计阅读 25 分钟

1. 复习 / Review: What Problem Does Sensitivity Analysis Solve?

Sensitivity analysis addresses one core question:
If our assumptions are slightly wrong, how much does our conclusion change?
如果假设偏差一点，结论会变多少？

本质 / Essence：把最优解 x* 看作参数 p 的函数（treat the optimal solution as a function of the parameter）， 然后用导数算出「参数变一点点，最优解变多少」。

2. 灵敏度系数的严格定义 / Formal Definition of Sensitivity

2.1 From absolute change to relative change / 从绝对变化到相对变化

Absolute change / 绝对变化：parameter p changes by Δp → optimal x changes by Δx

Relative change / 相对变化：p changes by Δp/p, x changes by Δx/x

Ratio / 比值：(Δx/x) / (Δp/p)
这个比值告诉你：p 每变化 1%，x 变化百分之几？

2.2 Take the limit to get the exact formula / 取极限得到精确公式

Let Δp → 0:
S(x, p) = (dx/dp) × (p/x)

• dx/dp — ordinary derivative (x's absolute rate of change with respect to p)
• Multiply by p/x — converts absolute rate to percentage rate (dimensionless!)

Why multiply by p/x? / 为什么乘 p/x？
Suppose your salary goes from 10,000 → 11,000 (+1,000 = +10%).
Your friend's goes from 100,000 → 101,000 (+1,000 = +1%).
Same absolute change, completely different relative significance.
同样的绝对变化，完全不同的相对意义。p/x 消除了量纲的影响。

3. 灵敏度分析的完整四步 / The Complete Four-Step Procedure

Step A: Treat the parameter as unknown (replace number with letter), re-derive x(p).
        将参数当作未知字母，重新推导 x(p)。

Step B: Differentiate to get dx/dp. / 对 x(p) 求导。

Step C: Evaluate S = (dx/dp) × (p/x) at the original parameter value.
        在原始参数值处计算 S。

Step D: Interpret sign and magnitude.
        解释符号和大小。
  • S > 0: p↑ → x↑ (same direction / 同向)
  • S < 0: p↑ → x↓ (opposite direction / 反向)
  • |S| < 0.5: robust / 稳健  | • |S| > 2: sensitive / 敏感

4. 完整示例 / Worked Example: Sensitivity to r (Price Falling Rate)

Step A / 推导 x(r)：
p = 0.65 − rt, other assumptions unchanged.
f(t) = (0.65 − rt)(200 + 5t) − 0.45t
    = −5rt² + (2.8 − 200r)t + 130
f′(t) = −10rt + (2.8 − 200r) = 0
→ x(r) = (2.8 − 200r) / (10r) = 0.28/r − 20

Domain check / 定义域：x(r) ≥ 0 ⟺ r ≤ 0.014

Step B / 求导：
dx/dr = −0.28/r² = −7/(25r²)

Step C / 计算 S：
At r = 0.01, x = 8:
dx/dr = −7/(25 × 0.0001) = −2800
S(x, r) = (−2800) × (0.01/8) = (−2800) × 0.00125 = −3.5

5. 第二个示例 / Worked Example: Sensitivity to g (Growth Rate)

Modify: w = 200 + gt (originally g = 5 lbs/day)
f(t) = −0.01gt² + (0.65g − 2.45)t + 130
f′(t) = −0.02gt + 0.65g − 2.45 = 0
→ x(g) = 32.5 − 122.5/g   (verify: g=5 → x=8 ✓)

dx/dg = 122.5/g², at g=5: dx/dg = 4.9
S(x, g) = 4.9 × (5/8) = 3.0625

6. Two Levels of Sensitivity / 两个层次的灵敏度

Decision sensitivity / 决策灵敏度 S(x, p)：参数变了，我该怎么做需要调整多少？
Objective sensitivity / 目标灵敏度 S(y, p)：参数变了，我能赚的目标值变多少？

They can be completely different! / 两者可能截然不同！
S(x, g) ≈ 3.06 (high) vs S(y, g) ≈ 0.17 (low).
增重速率判断错了 → 卖的时间偏差较大 → 但实际利润差不了多少！

Target sensitivity derivation / 目标灵敏度推导：
y(g) = 150.0625/g + 10.5625g + 50.375
dy/dg = 4.56 at g=5 → S(y, g) = 4.56 × (5/133.2) = 0.171
Retrieval Practice / 检索练习

Q1. Why do we use S = (dx/dp)×(p/x) instead of just dx/dp?

Q2. What does |S| = 0.2 tell you vs |S| = 3.5?

Q3. S(x, g) = 3.06 but S(y, g) = 0.17. What insight does this provide?

Key Takeaways / 本课要点

  • S(x, p) = (dx/dp)·(p/x): dimensionless ratio of relative changes
  • Complete 4-step procedure: derive x(p) → differentiate → compute S → interpret
  • Decision sensitivity S(x,p) ≠ Objective sensitivity S(y,p) — analyze both
  • |S| > 2 → parameter must be precise; |S| < 0.5 → rough estimate OK
  • Always check the domain of validity of x(p) / 检查 x(p) 的有效定义域
