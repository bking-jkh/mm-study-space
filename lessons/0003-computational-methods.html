0003 · 计算优化方法
Computational Methods for Optimization

基于 Lecture 3 (Lin, JNU) & Meerschaert §3.1–3.3 · 预计阅读 35 分钟

1. 为什么需要计算方法？/ Why Do We Need Computational Methods?

The plain and simple fact is that most equations cannot be solved analytically. / 绝大多数方程无法解析求解。

Nonlinear pig growth example / 非线性猪增长例子：
dw/dt = c·w, w(0) = 200 → w = 200e0.025t
f′(x) = (130c − 2cx)ecx − 0.45 = 0
→ This equation has no closed-form solution / 无法手算解出。
The two-stage strategy / 两阶段策略（贯穿整个数值求解的范式）：
① Global method / 全局方法：graphing or random search → locate the approximate region of the optimum
② Local method / 局部方法：Newton's method or golden section → refine to desired accuracy

Global methods are slow but robust; local methods are fast but need a good starting point.
全局方法慢但稳健，局部方法快但需要好初值。两者配合使用。

2. 图形法 / Graphical Method — The Simplest Global Method

f(x) = (0.65 − 0.01x)(200e0.025x) − 0.45x (x ≥ 0)

First plot (x: 0→20) / 第一轮画图：peak appears near x ≈ 19–20
First zoom-in / 第一次放大：peak at x ≈ 19.5
Second zoom-in / 第二次放大：x ≈ 19.47, f(x) ≈ 139.395

Final estimate / 最终估计：x* ≈ 19.5

Pros / 优点：intuitive, no derivatives needed, gives global perspective
Cons / 缺点：low accuracy (each decimal costs 10× work), cannot go beyond 2–3 dimensions
Complete graph / 完整图：A complete graph shows all important features of f over the entire domain S — all extrema, zeros, and asymptotic behavior. Don't just plot a small window and miss the big picture.

3. 牛顿法 / Newton's Method ⭐ 最重要的局部方法

3.1 Core idea / 核心思想：用切线代替函数 / Replace function with tangent line

We need to solve F(x) = 0 (e.g., F(x) = f′(x) for optimization).

Step 1 / 第一步：在 x₀ 处做切线 / Draw tangent at x₀
Tangent line equation / 切线方程：y = F(x₀) + F′(x₀)(x − x₀)

Step 2 / 第二步：找切线与 x 轴的交点 / Find where tangent crosses x-axis
Set y = 0: 0 = F(x₀) + F′(x₀)(x − x₀)
→ x₁ = x₀ − F(x₀)/F′(x₀)

Step 3 / 第三步：重复！/ Repeat!
x_{n+1} = x_n − F(x_n)/F′(x_n)
Geometric intuition / 几何直觉：
Imagine you're on a hillside at x₀, wanting to reach the valley bottom (where F(x)=0).
You look at the slope at your feet (F′(x₀)), extrapolate linearly to where the ground would hit sea level,
walk there (x₁), and reassess. Each step gets you closer.

Why is it fast? / 为什么快？
The tangent line is the best linear approximation (1st-order Taylor expansion).
Near a root, Newton's method converges quadratically — effective digits double each iteration!

3.2 Newton's method for optimization / 牛顿法用于优化

To optimize f(x), we solve f′(x) = 0. Set F(x) = f′(x):
x_{n+1} = x_n − f′(x_n)/f″(x_n)

Requires second derivative f″(x) — must be computable and non-zero near the root.

3.3 Pseudocode / 伪代码

Input: x₀ (initial guess), N (max iterations), ε (tolerance)
For n = 0 to N−1:
    x_{n+1} = x_n − F(x_n) / F′(x_n)
    If |x_{n+1} − x_n| < ε → return x_{n+1}
Output: x_N

⚠️ Fails if / 失败情况：F′(x_n) ≈ 0 (tangent parallel to x-axis → division by zero),
or if x₀ is too far from the root (may diverge).

3.4 Application / 应用：非线性猪问题

F(x) = (130c − 2cx)ecx − 0.45 = 0, c = 0.025
Graphical method gives x₀ = 19.5 / 图形法给出初值
Newton iterates 10× → converges to x ≈ 19.468

Sensitivity via Newton / 用牛顿法做灵敏度：
c=0.025 (baseline) → x=19.47; c=0.02525 (+1%) → x=20.02 (+2.84%)
→ S(x, c) ≈ 2.84

4. 黄金分割搜索 / Golden Section Search

For unimodal functions (single peak/valley). Requires no derivatives. / 适用于单峰函数，不需要导数。

Golden ratio / 黄金比例：φ = (√5 − 1)/2 ≈ 0.618
1 − φ ≈ 0.382

Test points / 测试点：
c = a + (1−φ)(b−a)  （左 38% 处）
d = a + φ(b−a)       （右 62% 处）

Decision rule (for a minimum) / 判断规则（求最小值）：
If f(c) < f(d): min is in [a, d] → set b = d
If f(c) ≥ f(d): min is in [c, b] → set a = c

Key advantage / 核心优势：each iteration only needs one new function evaluation — the other test point is inherited!
每次迭代只需一个新函数值，另一个复用！
