0007 · 线性回归模型
Linear Regression Model

基于 Lecture 7 (Lin, JNU) · 预计阅读 30 分钟

1. What is regression? / 什么是回归？

Given data points (xₜ, yₜ), find the best-fitting straight line y = β₀ + β₁x.
给一堆数据点，找一条最佳拟合直线。

General model / 一般模型：
y = f(X₁,…,Xₖ, β₀,…,βₖ) + ε
• y: dependent variable / 因变量 (random / 随机)
• Xⱼ: independent variables / 自变量 (non-stochastic / 非随机，由实验者控制)
• βⱼ: parameters / 参数 (to be estimated / 待估计)
• ε: random error / 随机误差 ~ i.i.d. N(0, σ²)

2. "Linear" means linear in parameters!

⭐ High-frequency exam point / 高频考点：
A model is linear if it is linear in the parameters β, NOT necessarily linear in X.
线性是对参数 β 而言，不是对自变量 X 而言！

Test / 判定方法：compute ∂y/∂βⱼ. If none of them contain any β → linear model.

✓ Linear: y = β₀ + β₁X₁² + β₂√X₂ + ε
  ∂y/∂β₁ = X₁² (no β), ∂y/∂β₂ = √X₂ (no β) → linear!

✗ Nonlinear: y = β₁²X₁ + β₂√X₂ + ε
  ∂y/∂β₁ = 2β₁X₁ (contains β₁!) → nonlinear

3. Least Squares Estimation / 最小二乘法

For simple linear regression: yₜ = β₀ + β₁xₜ + εₜ, t = 1,…,T.

Residual Sum of Squares (RSS) / 残差平方和：
S(β₀, β₁) = Σₜ₌₁ᵀ (yₜ − β₀ − β₁xₜ)²

Normal equations / 正规方程（令偏导为零）：
∂S/∂β₀ = −2Σ(yₜ − β₀ − β₁xₜ) = 0 → T·β₀ + β₁·Σxₜ = Σyₜ
∂S/∂β₁ = −2Σ(yₜ − β₀ − β₁xₜ)·xₜ = 0 → β₀·Σxₜ + β₁·Σxₜ² = Σyₜxₜ

Solution / 解：
β̂₁ = [T·Σxₜyₜ − Σxₜ·Σyₜ] / [T·Σxₜ² − (Σxₜ)²]
β̂₀ = ȳ − β̂₁·x̄   (ȳ = Σyₜ/T, x̄ = Σxₜ/T)

Hessian verification / Hessian 验证：
∂²S/∂β₀² = 2T > 0,   det(H) = 4T·Σ(xₜ−x̄)² > 0 (if not all xₜ are equal)
→ H positive definite → S is strictly convex → unique global minimum! / 唯一全局最小！

4. Model Evaluation / 模型评估 — R²

Total variation decomposition / 总变差分解：
SST = SSR + SSE
Σ(yₜ−ȳ)² = Σ(ŷₜ−ȳ)² + Σ(yₜ−ŷₜ)²
Total = Explained by regression + Residual

R² = SSR / SST ∈ [0, 1]: proportion of variation explained by the model.

⚠️ High R² ≠ good model (overfitting possible). Low R² ≠ bad model (data may be inherently noisy).
