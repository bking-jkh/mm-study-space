0006 · 梯度与凸优化基础
Gradient, Convex Functions & FPPA

基于 Lecture 4/6 (Lin, JNU) · 预计阅读 25 分钟

1. Directional Derivative & Gradient / 方向导数与梯度

Directional derivative / 方向导数：
Dᵤf(x) = lim_{t→0⁺} [f(x + t·u) − f(x)] / t
Rate of change of f at x in the direction of the unit vector u.

Gradient / 梯度：
∇f(x) = (∂f/∂x₁, …, ∂f/∂xₙ)ᵀ

Key relationship / 核心关系：Dᵤf(x) = ⟨∇f(x), u⟩
Directional derivative = inner product of gradient and direction.
• ∇f points in the direction of steepest ascent / 最速上升方向
• −∇f points in the direction of steepest descent / 最速下降方向 → 梯度下降法

2. Convex Functions / 凸函数 — The Optimizer's Best Friend

Convex set / 凸集 E：for any x,y ∈ E, λ∈(0,1), the point λx+(1−λ)y is also in E.

Convex function / 凸函数 f：for any x,y ∈ E, t∈(0,1),
f(tx + (1−t)y) ≤ t·f(x) + (1−t)·f(y)
Graph lies below the chord / 图形在弦的下方（碗形）。

First-order characterization / 一阶刻画（可微时）：
f is convex ⟺ f(y) ≥ f(x) + ⟨∇f(x), y−x⟩ for all x,y
Graph lies above every tangent plane / 图形在任意切平面的上方。

Second-order characterization / 二阶刻画：
f is convex ⟺ ∇²f(x) is positive semi-definite for all x
(All eigenvalues ≥ 0)

3. Why convexity matters / 为什么凸性如此重要？

Fermat's Rule for convex functions / 凸函数的 Fermat 规则：
For a differentiable convex function f on a convex open set E:
x* is a global minimizer ⟺ ∇f(x*) = 0

Note the double arrow (⟺)! For general functions, ∇f=0 is only necessary (⇒).
For convex functions, it's necessary and sufficient (⟺).
任何驻点就是全局最小点！一阶条件既是必要也是充分的。

Proof sketch:
(⇐) If ∇f(x*)=0, convexity gives f(y) ≥ f(x*) + ∇f(x*)ᵀ(y−x*) = f(x*) for all y. ✓
(⇒) If x* is a minimizer, all directional derivatives are ≥ 0, forcing ∇f(x*)=0.

4. FPPA / 不动点邻近算法 (Fixed-Point Proximity Algorithm)

For problems: minₓ f(x) + g(x) where f is smooth, g is non-smooth (e.g., ‖x‖₁).

Proximity operator / 邻近算子：
prox_{γg}(z) = argminₓ { g(x) + 1/(2γ)·‖x−z‖² }

FPPA iteration / FPPA 迭代：
x^{k+1} = prox_{γg}(x^k − γ∇f(x^k))
Step 1: gradient descent on f / 沿 f 的负梯度走一步
Step 2: proximity correction for g / 用邻近算子处理 g 的非光滑性
